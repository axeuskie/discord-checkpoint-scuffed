# Requirements

- Python
- Your Discord package.zip **WITH ONLY MESSAGES CHECKED**

(I didn't check with anything else, probably broken)

How to get it : https://support.discord.com/hc/en-us/articles/360004027692-Requesting-a-Copy-of-your-Data

# Installation

- Unzip `package.zip` in a _folder_
- Download `main.py` and place it in this _folder_
- Run `main.py`
- Enter a specific year you're interested in
